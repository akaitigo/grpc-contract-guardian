# 品質チェックリスト（全プロジェクト共通）

リリース前に全て満たすこと。例外はADRで明示的に除外する。

## 必須5項目

- [ ] READMEのクイックスタートがコピペで動く
- [ ] デモGIF/動画がREADME冒頭にある
- [ ] ハッピーパスのテストが1つ以上ありCIで通る
- [ ] 不可逆な判断のADRが1つ以上ある
- [ ] 未使用コード・TODO・console.log/printが消えている

## 検証コマンド

```bash
# テスト通過
make test

# lint/型チェック通過
make lint

# 全チェック一括
make check

# TODO/print の残存確認
grep -rn "TODO\|fmt\.Println\|fmt\.Print(" internal/ cmd/ --include="*.go"
```

## セキュリティチェック（AI生成コード必須）

> **警告: AI生成コードの36-40%にセキュリティ脆弱性が含まれる（業界調査）。**
> 以下のチェックを全PRに対して実行すること。

- [ ] SQLインジェクション: パラメータバインドを使用（文字列結合禁止）
- [ ] 認証・認可: 全エンドポイントに適切なアクセス制御がある
- [ ] 機密情報: ハードコードされたAPIキー・パスワードがない
- [ ] 依存関係: 既知の脆弱性を持つパッケージがない（`govulncheck ./...`）
- [ ] 入力バリデーション: 外部入力は全てバリデーション済み
- [ ] エラーハンドリング: スタックトレースがユーザーに露出しない

```bash
# セキュリティ脆弱性スキャン
govulncheck ./...

# 機密情報の漏洩チェック
grep -rn "sk-\|pk_\|ghp_\|AKIA\|password\s*=\s*[\"']" internal/ cmd/ --include="*.go"
```

## AI Agent Anti-patterns検知

- [ ] **コード省略パターン検知**: `// ... rest of code` / `// ... remaining implementation` の禁止
- [ ] **Hallucination Check**: import先がgo.modに実在するか検証
- [ ] **過剰ボイラープレート**: 同一パターンの3回以上繰り返しはリファクタリング対象

```bash
# 省略パターン検知
grep -rn "// \.\.\." internal/ cmd/ --include="*.go"
```

## 判定基準

| 項目 | 合格 | 不合格 |
|------|------|--------|
| クイックスタート | 新規clone→手順通りで動く | 暗黙の前提がある |
| デモ | 30秒以内で何ができるか分かる | テキストのみ |
| テスト | CI green、ハッピーパス網羅 | テストなし or red |
| ADR | 技術選定・アーキテクチャ判断が記録済み | 口頭のみ |
| クリーンコード | grep で残存ゼロ | TODO/debug出力が残存 |
| セキュリティ | 脆弱性スキャン通過、機密情報漏洩なし | AI生成コードを未チェックでマージ |
